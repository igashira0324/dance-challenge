2 自動セットアップ実行

同梱のバッチファイルを実行するだけで、リポジトリの初期化からプッシュまで完了します。

setup_git.bat

注意: 実行中にGitHubのログイン画面（ブラウザ）が表示された場合は、ログインを完了させてください。

3 手動で行う場合の手順

コマンドプロンプトやPowerShellで以下のコマンドを順に実行します。

  1. リポジトリ初期化: git init
  2. 全ファイル追加: git add .
  3. コミット作成: git commit -m "Initial commit"
  4. リモート登録: git remote add origin https://github.com/igashira0324/dance-challenge.git
  5. プッシュ: git push -u origin main
