1. 新世代ベクトル判定システム

従来のオイラー角ベースの判定から、3Dベクトル類似度（Cosine Similarity）ベースに移行しました。 これにより、ユーザーのカメラからの距離や角度に依存せず、極めて正確なポーズマッチングが可能になりました。

2. 座標系の同期 (Ripple Marker)

ビデオのミラー表示 (CSS scale-x[-1]) に合わせ、判定空間を自動的に反転させています。

dist = sqrt((1.0 - hand.x - marker.x)^2 + (hand.y - marker.y)^2)

3. シミュレーションとレンダリングの最適化

SkeletonUtils.clone を活用し、VRMモデルのロード時間をゼロに短縮。 ガイド用のシルエット（左右から飛んでくるモデル）は、ユーザーのポーズ定義から直接ボーン回転を生成しています。
