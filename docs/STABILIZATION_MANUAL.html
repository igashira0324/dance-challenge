1. 3Dポーズ判定エンジン

従来のオイラー角ベースから、体ローカル座標系における3D単位ベクトルによる判定へ移行しました。これにより、カメラからの距離や身長に関わらず安定した判定が可能になりました。

ミラー補正 (Mirroring)

Alt+Cでのキャプチャ時および実行時に一貫したミラー補正（X軸反転）を適用。ユーザーが鏡を見ている感覚で直感的に踊れるよう最適化されています。

ベクトル特徴抽出

extractPoseFeatures により、上腕と前腕の方向を正規化。アバターの骨格構造に依存しない汎用的な判定を実現しています。

2. リズム・スコアリング同期

ダンエボのような「ノリ」を再現するため、ポーズ精度だけでなくタイミング（Accuracy）を重視した設計になっています。

PERFECT!

タイミング・ウィンドウ

ヒット時刻から ±0.6秒を判定窓とし、PERFECTは即時フィードバック。GOODはウィンドウ終了までPERFECTへの昇格チャンスを維持します。

アスペクト比補正

Ripple判定において、動画の解像度（16:9等）に合わせてX軸の距離を補正。画面端のマーカーでも円形の判定を保ちます。

3. 実行環境 (Windows)

本エンジンの安定版を実行するには、以下のバッチファイルを使用してください。

@echo off
echo Launching AI Dance Challenge Stabilized...
npm run dev

※ Alt+C を押すことで、現在のポーズをJSON形式でコンソール出力し、楽曲データへ組み込むことができます。
