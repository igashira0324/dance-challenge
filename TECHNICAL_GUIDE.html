コアテクノロジー

Real-time Pose Tracking

MediaPipe PoseLandmarker を使用し、ブラウザ上でGPUアクセラレーションを効かせた高速な骨格検出を実現。Kalidokit による IK 補正で VRM モデルに自然な動きを付与します。

Dynamic Typography

TextAlive からインスパイアされた、BGM同期型歌詞レンダリング。Framer Motion と CSS Animations を組み合わせ、10種類のアニメーションパターンをリアルタイムに切り替えます。

Web Audio Engine

外部音源に頼らず、Web Audio API で高品質なシンセサイザー音を生成。120BPMのグリッドに基づいた正確な判定ロジックを搭載しています。

実行方法

Windows環境での起動は、プロジェクトルートにある run.bat をダブルクリックしてください。
