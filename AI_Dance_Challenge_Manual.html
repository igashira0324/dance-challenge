1. 概要

このアプリケーションは、Webカメラを使用してユーザーのポーズを認識し、3Dアバター（VRM）をリアルタイムで動作させるリズムゲームです。

2. 起動方法

プロジェクトのルートディレクトリにある start_game.bat をダブルクリックしてください。

.\start_game.bat

ブラウザで http://localhost:5173 が開きます。

3. アーキテクチャ図

graph TD A[App.tsx] --> B[PoseService] A --> C[AudioEngine] A --> D[VRMService] B --> B1[MediaPipe Pose] C --> C1[Web Audio API] D --> D1[Main VRM Model] D --> D2[Silhouette Clone L] D --> D3[Silhouette Clone R] A --> E[HUD] A --> F[KineticTypography]

4. 新機能: Dance Evolutionスタイル 3Dシルエット

ターゲットポーズのタイミングを知らせるため、プレイヤーのVRMモデルのクローンが半透明の緑色シルエットとして両サイドから接近します。シルエットが中央のプレイヤーモデルに合致した瞬間にポーズを合わせることで、高得点（PERFECT / GOOD）を獲得できます。

5. 技術スタック

  • React + Vite: 高速な開発環境とUI構築
  • MediaPipe Tasks: AIによる骨格検出
  • Three.js / @pixiv/three-vrm: 3Dアバターのレンダリング及びシルエット生成
  • Framer Motion: 滑らかなUIアニメーション
  • Lucide React: プレミアムなアイコンセット
